Show recent or historic activity per user or per site from any Stack Exchange site.

== Description ==

Create widgets for any Stack Exchange site, display questions or answers by specific users or generic for the whole site. The random mode will show posts from different sites at each page view.

Check the twin plugin [All Your Stack Posts](http://wordpress.org/plugins/all-your-stack-posts).

Translations: Espa√±ol.

== Installation ==

Extract the zip file and upload the contents to the `wp-content/plugins/` directory of your WordPress installation, and then activate the plugin from plugins page. 

= Uninstall =
If you delete the plugin, the entry will be deleted automatically via `unsinstall.php`.
